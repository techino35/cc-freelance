# cc-freelance

フリーランスエンジニア専用の Claude Code カンパニープラグイン。

案件獲得・顧客対応・開発・経理・リサーチを仮想会社組織として管理します。

## インストール

Claude Code で以下を実行:

```bash
/plugin marketplace add [your-github-username]/cc-freelance
/plugin install freelance@cc-freelance
```

## 起動

```bash
/freelance
```

初回は秘書がオンボーディングを開始します。
2回目以降は運営モードで自動起動します。

## 組織構成

```
あなた → 秘書室（窓口） → CEO（振り分け） → 各部署
```

| 部署 | 役割 |
|------|------|
| 秘書室 | TODO管理・壁打ち・日次管理（常設） |
| CEO | 振り分け・意思決定（常設） |
| レビュー | 週次・月次振り返り（常設） |
| 案件獲得 | 応募・提案・スコアリング・受注管理 |
| 顧客対応 | 既存クライアントの稼働・コミュニケーション管理 |
| 開発 | 実装タスク・設計・デバッグ管理 |
| 経理 | 収入・請求・目標管理 |
| リサーチ | 案件・技術・市場調査 |

## 使い方例

```
/freelance

秘書: おはようございます！今日のTODOを確認しますか？

> Upworkに新しいFastAPI案件を応募したい
秘書: 承知しました。案件スコアリングを実施してから提案書を作成します。
  → 案件獲得: スコアリング実施
  → 案件獲得: 提案書作成

> 今月の収入どれくらい？
秘書: 経理部に確認します。
  → 経理: 今月の収入サマリーを表示

> OliveKKの橋本さんからGASの修正依頼が来た
秘書: 承知しました。
  → 顧客対応: OliveKKの稼働ログに記録
  → 開発: 実装タスクを作成

> ダッシュボード
秘書: [全部署の概要を表示]
```

## スコアリング基準（案件評価）

| 評価軸 | 配点 |
|-------|------|
| 技術適合性（GAS/Python/FastAPI/AI API） | 4点 |
| 単価（¥2,000/h以上 = 満点） | 3点 |
| 稼働量（週20h以下 = 満点） | 2点 |
| クライアント信頼性（評価4.5以上） | 1点 |

- 7点以上 → 即応募
- 5〜6点 → 条件次第
- 4点以下 → 見送り
