Privacy Policy — JSON Viewer & Editor

Last updated: March 30, 2026

1. Overview

JSON Viewer & Editor ("the Extension") is a Chrome browser extension that automatically formats JSON content displayed in the browser. This Privacy Policy explains what data is accessed and how it is handled.

2. Data Collection

This extension does not collect, store, transmit, or share any personal data or user information. All processing occurs entirely within your local browser.

3. Permissions Used

  • storage: Used solely to save your UI preferences (e.g., dark/light mode setting) locally in your browser. This data never leaves your device.
  • Host permissions (*://*/*): Required to detect JSON content on any webpage or API endpoint URL you navigate to. The extension reads page content only to identify and format JSON responses. No data is sent to any server.

4. No Network Requests

This extension makes no network requests. It does not communicate with any external servers, APIs, or third-party services.

5. Data Sharing

We do not sell, share, or transfer any user data to third parties.
