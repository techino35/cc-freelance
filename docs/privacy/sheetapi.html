Privacy Policy — SheetAPI

Last updated: March 23, 2026

1. Overview

SheetAPI ("the Add-on") converts your Google Sheets into a REST API endpoint, allowing external applications to read and write spreadsheet data via HTTP requests. This Privacy Policy explains what data is accessed and how it is handled.

2. Data We Collect and Access

Data TypePurpose
Spreadsheet cell valuesExpose as JSON via API endpoint / write data from API requests
API keys you generateAuthenticate incoming API requests; stored in PropertiesService
Request logs (optional)Track API usage, stored within your spreadsheet

3. How We Use Your Data

  • Spreadsheet data is served as JSON in response to authenticated HTTP requests that you authorize.
  • API keys are stored locally in PropertiesService and used solely for request authentication.
  • The developer does not receive or store your spreadsheet data or API keys.

4. Data Sharing

We do not sell, rent, or share your data with any third parties. Data is exposed only to callers that possess the API key you generate and control.

5. Data Storage

API keys and configuration are stored in Google Apps Script's PropertiesService. No data is stored on servers operated by the developer.

6. Security

All API endpoints are protected by API key authentication. Keys should be kept confidential. You can revoke and regenerate keys at any time from the add-on settings.

7. Google API Services User Data Policy

SheetAPI's use of information received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements.
