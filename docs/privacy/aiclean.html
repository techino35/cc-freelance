Privacy Policy — AIClean

Last updated: March 23, 2026

1. Overview

AIClean ("the Add-on") is a Google Sheets add-on that uses AI to clean, normalize, and standardize data in your spreadsheets. This Privacy Policy explains what data is accessed and how it is handled.

2. Data We Collect and Access

Data TypePurpose
Spreadsheet cell values (selected range)Send to AI API for cleaning/normalization processing
User email addressLicense key association
API key and settingsStored locally via PropertiesService

3. AI Processing

Selected cell data is sent to an AI API (such as OpenAI or Google Gemini) for processing. By using AIClean, you acknowledge that selected data will be transmitted to the configured AI service. Please review the privacy policy of the AI provider you configure:

  • OpenAI Privacy Policy
  • Google Privacy Policy

4. Data Sharing

We do not sell, rent, or share your data with any third parties beyond the AI processing request you initiate. The developer does not receive or store your spreadsheet data.

5. Data Storage

Before cleaning, a backup of the original data is stored within your spreadsheet. AI API keys and settings are stored in Google Apps Script's PropertiesService. No data is stored on servers operated by the developer.

6. Google API Services User Data Policy

AIClean's use of information received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements.
