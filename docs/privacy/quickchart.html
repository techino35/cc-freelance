Privacy Policy — QuickChart for Google Sheets

Last updated: March 23, 2026

1. Overview

QuickChart for Google Sheets ("the Add-on") generates chart images from your selected spreadsheet data using the quickchart.io rendering service. This Privacy Policy explains what data is accessed and how it is handled.

2. Data We Collect and Access

Data TypePurpose
Selected cell range values (labels and numbers)Construct chart configuration for rendering
User email addressLicense key association only
License key and usage countersStored locally via PropertiesService

3. Third-Party Chart Rendering

Chart data (cell values and styling options) is sent to quickchart.io via HTTPS to render the chart image. This data is not retained by quickchart.io beyond the duration of the render request. Please review quickchart.io's privacy policy for details.

4. Data Sharing

We do not sell, rent, or share your data with any third parties beyond the chart rendering request to quickchart.io described above.

5. Data Storage

License data and usage counters are stored exclusively in Google Apps Script's PropertiesService. Spreadsheet data is never stored by the developer.

6. Google Drive Access

When you use the "Save to Drive" output option, the Add-on creates PNG files in a folder named "QuickChart" in your Drive. The drive.file scope limits access to only files created by this add-on — it cannot access any other Drive files.

7. Google API Services User Data Policy

QuickChart's use of information received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements.
