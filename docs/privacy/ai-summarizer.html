Privacy Policy — AI Page Summarizer

Last updated: March 30, 2026

1. Overview

AI Page Summarizer ("the Extension") is a Chrome browser extension that summarizes webpage content using the Anthropic Claude AI API. This Privacy Policy explains what data is accessed and how it is handled.

2. Data We Access

Data TypePurpose
Webpage text content (active tab)Sent to Anthropic Claude API to generate a summary
Your Anthropic API keyStored locally via chrome.storage.sync; used to authenticate API requests
Summary history (last 20 entries)Stored locally via chrome.storage.local; never leaves your device

3. Third-Party Services

Page content is sent to the Anthropic Claude API (api.anthropic.com) over HTTPS to generate summaries. Your API key is included in request headers as required by Anthropic. Please review Anthropic's Privacy Policy for details on how they handle data.

4. Data the Developer Does Not Collect

The extension developer does not collect, store, or transmit any user data. No analytics, no telemetry, no external servers operated by the developer.

5. Permissions Used

  • activeTab: Required to read the text content of the currently active tab when you click "Summarize This Page".
  • storage: Used to save your API key and summary history locally in your browser.
  • scripting: Required to extract page text content from the active tab.

6. Data Sharing

We do not sell, rent, or share your data with any third parties beyond the API request to Anthropic described above.
