Privacy Policy — SheetDiff

Last updated: March 23, 2026

1. Overview

SheetDiff ("the Add-on") is a Google Sheets add-on that compares snapshots of your spreadsheet data and highlights changes between versions. This Privacy Policy explains what data is accessed and how it is handled.

2. Data We Collect and Access

Data TypePurpose
Spreadsheet cell values (full sheet or selected range)Create snapshots and compare changes between versions
User email addressAssociate snapshots with the user who created them
Snapshot dataStored in a hidden sheet within your spreadsheet

3. How We Use Your Data

  • Cell data is read to create snapshots, which are stored within your own spreadsheet (not on external servers).
  • Diff results are displayed in-sheet via cell highlighting and a report sheet.
  • The developer does not receive or store any of your spreadsheet data.
  • No data is transmitted to external servers.

4. Data Sharing

We do not sell, rent, or share your data with any third parties.

5. Data Storage

Snapshots are stored as hidden sheets within your own Google Spreadsheet. Configuration settings are stored in Google Apps Script's PropertiesService. No data is stored on servers operated by the developer.

6. Google API Services User Data Policy

SheetDiff's use of information received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements.
